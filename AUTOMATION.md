# 🤖 Автоматизація Google Apps Script

## ✅ Що АВТОМАТИЗОВАНО через CLI

### Команда: `npm run setup-webhook`

Ця команда автоматично:

1. **✅ Відкриває Google Sheets** у вашому браузері
2. **✅ Копіює готовий код** в буфер обміну
3. **✅ Генерує WEBHOOK_SECRET** (вже вставлено в код)
4. **✅ Створює налаштований файл** `webhook-configured.gs`
5. **✅ Показує покрокові інструкції**

## ❌ Що НЕМОЖЛИВО автоматизувати

Google Apps Script API має обмеження безпеки:

- ❌ **Автоматичний деплой скрипта** - потрібна OAuth2 авторизація через браузер
- ❌ **Автоматичне створення тригерів** - потрібен ручний дозвіл користувача
- ❌ **Пряме редагування через API** - вимагає складної авторизації

## 🚀 Як це працює зараз

### Крок 1: Запустіть автоматизацію

```bash
npm run setup-webhook
```

**Що відбувається:**
- ✅ Відкривається Google Sheets
- ✅ Код копіюється в буфер обміну
- ✅ Показуються інструкції

### Крок 2: 3 ручні дії (30 секунд)

У відкритій Google Sheets:

1. **Extensions** → **Apps Script**
2. **Видалити код** → **Вставити** (Ctrl+V)
3. **Налаштувати тригер:**
   - Значок ⏰
   - + Add Trigger
   - Function: `onEdit`
   - Event type: `On edit`
   - Save

### Крок 3: Готово! 🎉

Тепер при будь-якій зміні в таблиці:
```
Google Sheets → onEdit() → Webhook → Vercel API → Сайт оновлюється
```

## 🔧 Технічні деталі

### Чому потрібна ручна авторизація?

Google Apps Script API вимагає:
1. OAuth2 Client Credentials
2. Інтерактивну авторизацію у браузері
3. Scope: `script.projects` (чутливі дані)

Це захисний механізм Google для запобігання зловживань.

### Альтернативні методи (які я спробував)

#### ❌ Метод 1: `clasp` CLI
```bash
npm install -g @google/clasp
clasp login
clasp create --type sheets
```
**Проблема:** Потрібна Node.js 20+, ручний login

#### ❌ Метод 2: Apps Script API
```javascript
const script = google.script({ version: 'v1', auth });
await script.projects.create({ ... });
```
**Проблема:** Потрібна OAuth2 авторизація

#### ✅ Метод 3: Готовий код + Автовідкриття (використовуємо)
```javascript
// Генеруємо код з правильними налаштуваннями
// Відкриваємо браузер
// Копіюємо в буфер обміну
```
**Перевага:** Максимально швидко, без складних налаштувань
